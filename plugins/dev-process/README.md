# dev-process

Development workflow guardrails: pre-implementation review, session retrospectives, codebase rule discovery, and GitHub Actions review.

## Commands

| Command | Description |
|---------|-------------|
| `/gather-rules` | Explore codebase for conventions and save as path-scoped claude rules |
| `/review-github-actions` | Lint and review GitHub Actions workflow files |

## Skills

| Skill | Description |
|-------|-------------|
| `pre-implementation-review` | Surface questions, assumptions, and edge cases before coding |
| `session-retrospective` | Extract learnings from session and persist to `.claude/rules/` |

## Configuration

No configuration needed. All commands and skills work with zero setup.

## Installation

```bash
claude plugin add dev-process@jbalatero
```
